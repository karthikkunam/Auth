### Prerequisites 

- Node 10 or higher

 #### Node

Installed node using brew. 
```bash
brew install node  
```
> If already installed, you can upgrade node `brew upgrade node`.

### Dependencies

Run the command
```bash
npm install
```
This should install all the dependencies for the project.

### Testing

To run all tests:
```bash
npm run test
```
